## 经营范围

### 唯一经营项目：家居日用

**产品分类（8大类）：**

1. **家居清洁用品**
   - 拖把、扫帚等清洁工具
   - 清洁剂、洗涤剂
   - 垃圾桶、垃圾袋
   - 除尘用品
   - 清洁刷具

2. **厨房日用品**
   - 餐具、碗筷
   - 厨房刀具
   - 切菜板
   - 厨房收纳用具
   - 厨房清洁用品

3. **收纳整理用品**
   - 收纳箱、收纳盒
   - 衣物整理袋
   - 置物架、储物架
   - 挂钩、粘钩
   - 抽屉分隔板

4. **洗衣晾晒用品**
   - 衣架、晾衣架
   - 洗衣篮、脏衣篓
   - 晾衣夹
   - 防尘罩
   - 衣物护理用品

5. **卫浴日用品**
   - 毛巾、浴巾
   - 牙刷杯、皂盒
   - 浴室置物架
   - 防滑垫
   - 浴室清洁用品

6. **家居小工具**
   - 手电筒、照明灯
   - 电池
   - 胶带、胶水
   - 简易工具
   - 家居维修小物

7. **家居塑料制品**
   - 塑料桶、水桶
   - 塑料盆、洗脸盆
   - 塑料凳子
   - 塑料收纳用品
   - 塑料日用容器

8. **其他家居日用**
   - 防蚊防虫用品
   - 除湿除臭用品
   - 雨具
   - 家庭应急用品
   - 季节性日用品
