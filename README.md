## GitHub Pages 部署

### 快速部署步骤

1. **创建GitHub仓库**
   ```
   - 登录 GitHub
   - 点击 "New repository"
   - 仓库名：zigong-official
   - 设为 Public
   ```

2. **上传文件**
   ```
   - 将所有文件上传到仓库
   - Commit message: "Initial commit"
   ```

3. **启用 Pages**
   ```
   - Settings → Pages
   - Source: Deploy from a branch
   - Branch: main / (root)
   - Save
   ```

4. **访问网站**
   ```
   https://你的用户名.github.io/仓库名/
   ```
