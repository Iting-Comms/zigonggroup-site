# 惠州市子贡贸易有限公司 官方网站

清洁卫生及厨房餐饮用品专业供应

## 项目说明

本网站是惠州市子贡贸易有限公司的官方企业展示网站，专注于：
- **清洁卫生用品**
- **厨房餐饮用品**

**仅此两项**，完全符合腾讯广告审核"家居日用"类别要求。

## 产品分类

### 1️⃣ 清洁卫生用品（3个子类）

#### 清洁工具系列
- 拖把系列：360度旋转拖把、平板拖把、胶棉拖把
- 扫除工具：软硬毛扫帚、防风簸箕
- 刷具类：马桶刷套装、瓷砖刷、玻璃刷

#### 清洁用品
- 清洁剂：多功能清洁剂、玻璃水、去油剂、洁厕灵
- 清洁辅助：超细纤维抹布、海绵擦、百洁布、钢丝球
- 垃圾袋：加厚型、点断式、多规格

#### 收纳整理用品
- 收纳箱盒：透明塑料箱（30L-150L）、滑轮款
- 置物架：不锈钢多层架、壁挂架
- 挂钩：强力粘钩、吸盘挂钩、S钩组合

### 2️⃣ 厨房餐饮用品（3个子类）

#### 餐具厨具系列
- 餐具：陶瓷碗碟套装、不锈钢筷勺、儿童餐具
- 刀具板具：不锈钢菜刀、多功能剪刀、切菜板
- 厨房小件：锅铲勺具、开瓶器、削皮器
- 收纳用具：刀架、筷笼、调味罐架、沥水架

#### 食品保鲜储存
- 保鲜盒：玻璃/塑料材质、微波炉适用
- 保鲜膜袋：PE保鲜袋、加厚保鲜膜
- 储存罐：透明密封罐、五谷杂粮罐、调料罐

#### 厨房清洁用品
- 洗涤用具：木纤维洗碗布、洗碗海绵、钢丝球
- 清洁用品：去油污剂、洗洁精、厨房湿巾
- 便利用品：厨房纸巾、吸油纸、一次性手套
- 垃圾处理：脚踏垃圾桶、感应垃圾桶、分类垃圾袋
